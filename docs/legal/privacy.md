## 제3조의2 (개인정보 파기 절차 및 방법)

회사는 보유 기간 경과·처리 목적 달성 등으로 개인정보가 불필요하게 된 경우 지체 없이 파기합니다 (개인정보보호법 제21조).

1. **파기 절차**: 회원 탈퇴·목적 달성으로 불필요해진 개인정보는 즉시 파기 대상으로 분류합니다. 법령상 보존 의무가 있는 정보(제3조 — 결제·청약철회 기록)는 별도로 분리 보관한 뒤 보존 기간이 경과하면 파기합니다.
2. **파기 방법**:
   - 전자적 파일 형태: 복구·재생이 불가능한 방법으로 영구 삭제합니다 (DB 레코드 삭제 + Cloudflare R2 객체 삭제). 재해복구용 기술 백업에 포함된 정보는 복구 목적 외로 이용하지 않으며 최대 30일 내 자동 파기됩니다.
   - 비전자적(출력물) 형태: 분쇄 또는 소각합니다 (현재 비전자적 보유분 없음).
3. **파기 시점**: 회원 탈퇴 시 즉시(법령상 보존 의무분 제외), 재해복구 백업은 30일 내, 분석·광고 데이터는 GA4 속성 보관 설정(2 또는 14개월)에 따릅니다.

## 제6조 (사용자 권리)

사용자는 다음 권리를 행사할 수 있습니다:

1. **열람 권리**: 본인 프로필·활동 기록은 서비스 내에서 직접 확인. 그 외 보유 개인정보 열람은 privacy@terraworld.app 로 요청(처리 후 회신).
2. **정정 권리**: 닉네임·이메일 변경(서비스 내). 그 외 정정은 위 이메일로 요청.
3. **삭제 권리**: 활동 기록 삭제(서비스 내) + 회원 탈퇴 시 계정·콘텐츠 즉시 삭제(법령상 보존 의무 데이터 제외 — 제3조).
4. **처리 정지 권리**: 회원 탈퇴(즉시 처리).
5. **데이터 이전(이동) 권리** (GDPR): 현재 자동 export 기능 **미제공** — 요청 시 수동 제공으로 갈음(JSON export UI 도입 예정).
6. **광고/분석 거부**: 광고 보상은 시청을 거부할 수 있음(보상 미지급). 앱 내 **프로필 > 동의 항목 관리** 에서 마케팅·분석·광고 동의를 직접 끄거나 켤 수 있으며, 단말 OS 의 "광고 추적 제한" 설정·iOS ATT·GA Opt-out add-on 으로도 거부 가능합니다.

GDPR 준수 사항 (EU 사용자 — 영어 서비스 도입 시):

- Data Protection Officer (DPO) 지정
- Data Subject Access Request (DSAR) 30일 내 응답
- 데이터 침해 시 72시간 내 감독기관 통보

## 제7조 (보안 조치)

회사는 다음 보안 조치를 적용합니다:

1. **암호화**: 비밀번호 scrypt 단방향 해시, JWT RS256 서명, 전송 구간 HTTPS(TLS 1.2 이상) 강제
2. **접근 통제**: better-auth 세션(HttpOnly·SameSite=Strict 쿠키, 7일), 인증·서비스 데이터 스키마 분리
3. **DB 보안**: PostgreSQL 16, DB 접근 권한 통제 및 정기 백업
4. **보안 헤더**: X-Frame-Options / X-Content-Type-Options / Strict-Transport-Security 등 적용
5. **취약점 점검**: CodeQL 정적 분석(매 PR + 정기 스캔)
6. **rate limiting**: Redis 기반 사용자·IP 게이트

## 제8조 (만 14세 미만 처리)

- [ ] **사업자 정보**: 회사명/대표자/사업자등록번호/주소/연락처 (제11조 + 약관 — placeholder 상태)
- [ ] **개인정보 보호책임자(CPO)** 지정: 이름·직책·연락처 (PIPA 제31조). 한국 문서 본문은 CPO 로 표기, GDPR DPO 는 EU 적용 + Art.37 요건 충족 시 별도 판단
- [ ] **시행일** 확정 + 시행 전 공고
- [ ] **마케팅·광고성 정보 수신동의** 별도 opt-in 흐름 (정보통신망법 제50조 — 현재 미구현. 푸시/이메일 마케팅 발송 시 사전 동의 + 야간 발송 제한)
- [ ] **게임물 등급분류(GRAC)** 해당 여부 (게임산업진흥법 — 가상재화+IAP. 해당 시 등급분류 없이 국내 유통 불가 → 별도 BLOCKING 가능, 변호사/GRAC 확인)
- [ ] **전자상거래법 통신판매업 신고** 의무 여부 + 청약철회(7일) 처리 주체(자체 vs Apple/Google) 정합
- [x] **위치정보 "수집 안 함" 명시**: 제9조의2 추가 완료. (향후 수집 시 위치정보법 별도 동의 필요 — 그 시점 사업/법무 결정)
- [ ] **GDPR 적용 범위**: 한국 한정 출시면 범위 명확화, EU 유입 시 적법근거/DSAR/72h 침해통보 실 운영
