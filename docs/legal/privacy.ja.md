## 第1条（収集・利用の目的）

当社は次の目的で個人情報を収集・利用します。

| 目的 | 項目 |
| --- | --- |
| 会員登録・認証 | メール、パスワード（scrypt 一方向ハッシュ）、ニックネーム |
| サービス提供 | 活動記録、仮想財（しずく・ひざし・トークン）、テラリウム配置、フレンド関係 |
| 決済 | 決済レシート（Play Billing / Apple IAP）、返金／撤回処理 |
| プッシュ通知 | FCM トークン（Firebase Cloud Messaging） |
| マーケティング・分析 | Google Analytics 4（仮名・集計分析、個人識別子は非送信）、広告識別子（AdMob リワード広告） |

## 第2条（収集項目＋必須／任意同意）

PIPA 第15条に基づき: 必須／任意の同意項目＋拒否権＋拒否時の不利益を明示。

### 2.1 会員登録時

| 項目 | 同意 | 拒否時の不利益 |
| --- | --- | --- |
| メールアドレス | **必須** | 登録不可（アカウント識別＋パスワード再発行） |
| パスワード（scrypt 一方向ハッシュ、平文は保存しない） | **必須** | 登録不可（認証手段） |
| ニックネーム | **必須** | 登録不可（サービス内のユーザー識別） |
| 生年月日（満14歳未満ブロックの検証用） | **必須** | 登録不可（PIPA 上の児童の個人情報 — 満14歳未満は法定代理人の同意なく処理不可、本サービスは登録をブロック） |

### 2.2 利用中（自動収集）

| 項目 | 同意 | 拒否時の不利益 |
| --- | --- | --- |
| 活動記録の本文（散歩・読書・ランニング・らくがき・カスタム） | **必須** | コア機能が利用不可 |
| 写真（Cloudflare R2 に保存） | **任意** | なし（写真添付なし、テキスト記録のみ可） |
| 端末情報（OS／ブラウザ／解像度） | **必須** | 互換性・セキュリティ（レート制限）が機能しない |
| 広告 ID（Android GAID / iOS IDFA） | **任意** | 広告のパーソナライズなし（一般広告を表示） |
| FCM トークン（プッシュ） | **任意** | プッシュ通知を受信しない（出席／枯れ／フレンド活動） |
| GA4 分析データ（仮名・集計） | **任意** | なし（**プロフィール > 同意項目の管理**で拒否可＋GA オプトアウト アドオン／端末トラッキング制限／iOS ATT） |

### 2.3 決済時（Play Billing / Apple IAP に委任）

| 項目 | 同意 | 拒否時の不利益 |
| --- | --- | --- |
| 決済レシート ID（purchaseToken） | **必須**（決済時） | 決済を進められない |
| 購入プロダクト ID | **必須**（決済時） | 決済を進められない |
| クレジットカード情報 | **収集しない** | — Google / Apple の決済システムに委任（当社は保有しない） |

### 2.4 拒否権の行使方法

- 登録時: 必須項目の同意を拒否すると登録が拒否されます。任意項目を拒否しても登録は進みます（該当機能は無効化）。
- 利用中: アプリ内**プロフィール > 同意項目の管理**でマーケティング・分析・広告の同意を直接撤回・再同意できます（または privacy@terraworld.app へ請求）。広告・分析は端末 OS 設定・ATT でも即時に拒否可能。
- 問い合わせ／同意撤回: privacy@terraworld.app（レビュー完了後に有効化）。

## 第3条（保有・利用期間）

| カテゴリ | 保有期間 |
| --- | --- |
| 会員情報（メール・ニックネーム） | 退会後ただちに削除 |
| 活動記録／テラリウム／フレンド関係 | 退会後ただちに削除 |
| 写真（R2） | 退会時に原本を即時削除。災害復旧用の技術バックアップは復旧目的外に使用せず、最大30日以内に自動破棄 |
| 決済記録 | 5年（電子商取引法） |
| 返金／撤回記録 | 3年（電子商取引法） |
| 広告／分析データ | GA4 プロパティの保持設定（2 または 14か月）に従う |
| アクセスログ | 3か月（情報通信網法） |

## 第3条の2（破棄の手続き・方法）

当社は、保有期間の経過・処理目的の達成等により個人情報が不要になった場合、遅滞なく破棄します（PIPA 第21条）。

1. **手続き**: 退会・目的達成により不要になった個人情報はただちに破棄対象に分類します。法令上の保存義務がある情報（第3条 — 決済・撤回記録）は別途分離保管し、保存期間の経過後に破棄します。
2. **方法**:
   - 電子ファイル: 復元・再生が不可能な方法で永久に削除します（DB レコード削除＋Cloudflare R2 オブジェクト削除）。災害復旧用バックアップに含まれる情報は復旧目的外に利用せず、最大30日以内に自動破棄します。
   - 非電子（印刷物）: 裁断または焼却します（現在、非電子の保有なし）。
3. **時期**: 退会時に即時（法令上の保存義務分を除く）、災害復旧バックアップは30日以内、分析・広告データは GA4 プロパティ保持設定（2 または 14か月）に従う。

## 第4条（第三者提供・共有）

当社は次の場合に限り個人情報を第三者に提供・共有し、それ以外では提供しません。

1. ユーザーが事前に別途同意した場合。
2. 法令に基づく適法な手続き（捜査機関の要請／裁判所の令状等）。
3. **広告識別子に基づく広告表示・測定（Google AdMob / AdSense）**: 広告 SDK 提供者が広告配信・不正クリック防止・成果測定のため広告識別子（GAID/IDFA）・端末/接続情報を処理し、提供者が自社の広告・測定目的でも活用しうるため、単なる委任ではなく**第三者提供（共有）**とみなします。詳細は [Google の広告ポリシー](https://policies.google.com/technologies/ads)。拒否: 端末 OS の「広告トラッキングの制限」（Android）／アプリのトラッキングの透明性（ATT, iOS）。

## 第6条（ユーザーの権利）

ユーザーは次の権利を行使できます。

1. **アクセス権**: 自身のプロフィール・活動記録はサービス内で直接確認。その他の保有個人情報の閲覧は privacy@terraworld.app へ請求（処理後に回答）。
2. **訂正権**: ニックネーム・メールの変更（サービス内）。その他の訂正は上記メールへ。
3. **削除権**: 活動記録の削除（サービス内）＋退会時にアカウント・コンテンツを即時削除（法令上の保存義務データを除く — 第3条）。
4. **処理停止権**: 退会（即時処理）。
5. **データポータビリティ権**（GDPR）: 現在、自動エクスポートは**提供していません** — 請求時に手動提供で代替（JSON エクスポート UI を導入予定）。
6. **広告／分析の拒否**: 広告は視聴を拒否できます（報酬なし）。アプリ内**プロフィール > 同意項目の管理**でマーケティング・分析・広告の同意をオン／オフでき、端末 OS の「広告トラッキングの制限」・iOS ATT・GA オプトアウト アドオンでも拒否できます。

GDPR 遵守事項（EU ユーザー — 英語サービス導入時）:

- データ保護責任者（DPO）の指定
- データ主体アクセス要求（DSAR）への30日以内の応答
- データ侵害時の72時間以内の監督機関への通知

## 第7条（セキュリティ対策）

1. **暗号化**: パスワード scrypt 一方向ハッシュ、JWT RS256 署名、転送区間 HTTPS（TLS 1.2 以上）を強制。
2. **アクセス制御**: better-auth セッション（HttpOnly · SameSite=Strict クッキー、7日）、認証／サービスデータのスキーマ分離。
3. **DB セキュリティ**: PostgreSQL 16、DB アクセス権限の統制と定期バックアップ。
4. **セキュリティヘッダー**: X-Frame-Options / X-Content-Type-Options / Strict-Transport-Security 等。
5. **脆弱性点検**: CodeQL 静的解析（毎 PR ＋定期スキャン）。
6. **レート制限**: Redis ベースのユーザー・IP ゲート。

## 第8条（満14歳未満の処理）

当社は満14歳未満の会員登録を受け付けません。満14歳未満と確認された場合、アカウント・データをただちに削除し、保護者に通知します。
